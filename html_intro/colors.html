Для того, чтобы задать используем style в котором прописываем изменения

Способы указания цветов:

HEX код

  1. background-color: (пишем номер цвета согласно таблице HEX) в нашем случае #1c87c9
  2. color: (пишем номер цвета согласно таблице HEX) в нашем случае: #d5dce8
  3. padding: (Выбираем размерки блока, который будет окрашен) в нашем случае 22px

RGB код

  1. background-color: rgb(25,25,112) (пишем номер цвета согласно таблице RGB)
  2. color: rgb(169,169,169) (пишем номер цвета согласно таблице RGB)
  3. padding: (Выбираем размерки блока, который будет окрашен)
