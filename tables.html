Таблицы в HTML

Для создания таблицы используется парный тег table, таблица создается рядами, которые обозначаются tr(сокр.от table row),
внутри которого прописывается информация под тегом td(сокр. от table data),
которая соотносится с заголовками в таблице, каждый столбец записывается отдельным td
Заголовок таблицы задается при помощи тега th(сокр. от table head)
тег th не является основным, но для правильности структурирования таблицы лучше всегда его использовать

Границы таблицы

Чтобы разграничить таблицу по полям, используется атрибут border к тегу table,
значение которого по умолчанию 1 пиксель, но в кавычках можно указать желаему толщину рамки
Также, разграничить таблицу можно при помощи CSS стилей путем изменения параметров table,th,td{border:1px solid #cccccc}
