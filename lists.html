Виды списков:

Маркированный список

  • Купить хлеб
  • Купить молоко
  • Купить колбасы

У элемента ul есть атрибуты, например square, который записывается в формате ul type="square", что приводит наш список к такому виду

  • Купить хлеб
  • Купить молоко
  • Купить колбасы

Также, в атрибут type меняется на style и можно использовать свойства css list-style-type или list-style-image

  • Купить хлеб
  • Купить молоко
  • Купить колбасы

Нумерованный список

Для оформления нумерованного списка используется ol

  1. Купить хлеб
  2. Купить молоко
  3. Купить колбасы

Также нумерованный список может быть пронумерован буквами, путем добавления type="A" в атрибуты

  1. Купить хлеб
  2. Купить молоко
  3. Купить колбасы

Или же нумерация римскими цифрами type="I"

  1. Купить хлеб
  2. Купить молоко
  3. Купить колбасы

Для создания такого типа списков используется тег dt, а чтобы дать определение используют тег dd
